# Devchallenge
<a href="https://devchallenge.now.sh/">DevChallenge</a> permite que você evolua suas skills como programador!

# Techs utilizadas:
HTML<br>
CSS<br>
Font Awesome<br>

# Requisitos:
- A página deve se parecer o mais próximo possível do design<br>
- A página deve ser responsiva<br>
